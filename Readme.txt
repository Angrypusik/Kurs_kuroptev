Алгоритм запуска приложения:
1)Сначала разворачивается докерфайл командой: docker build -t traffic .
2)Далее запускается докерфайл командой docker run -p 8501:8501 traffic
3)Открывается ссылка на веб-приложение, в котором можно загрузить любой видеоролик, также можно вручную открыть сайт прописав в адресной строке: localhost:8501
4)После загрузки видео оно начинает обрабатываться нейросетью
5)В итоге на сайт выводится обработанное моделью видео, которое можно просмотреть и скачать.
